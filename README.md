# infinitode-2-automation
Infinitode 2 Game automation using PYTHON and ADB <br>
Why? <br>
Can't use PYAUTOGUI because it doesn't work android. <br>
So I had to use PYTHON and ADB. <br>


##### craft_tiles.py : It automatically converts/crafts low level tiles to higher level tiles.


#### steps to use:
1) Download and/or install ADB on your pc.
2) Add ADB to the environment variable(optional).
3) Connect your phone using adb. Both wired and wireless connection is supported.
4) Run the game.
5) Run the script and watch the magic happen!
